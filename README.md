# KVN REST API

## Project Overview
This project is a REST API application for managing a KVN competition system.  
It handles **teams**, **games**, and **results**, and is built using **FastAPI**, **SQLAlchemy**, **Alembic**, and **PostgreSQL**.

## Domain Model
- **Team** — KVN teams
- **Game** — KVN games
- **Result** — team performance in a game

## Tech Stack
- Python 3.10  
- FastAPI  
- SQLAlchemy  
- Alembic  
- PostgreSQL  
- Uvicorn
